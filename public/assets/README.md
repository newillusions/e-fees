# Public Assets Directory

This directory contains static assets that will be served directly by the web server.

## Favicon Files
Place your favicon files here:
- `favicon.ico` - Traditional favicon (16x16, 32x32, 48x48)
- `favicon.svg` - SVG favicon (modern browsers)
- `apple-touch-icon.png` - Apple touch icon (180x180)
- `favicon-16x16.png` - 16x16 PNG favicon
- `favicon-32x32.png` - 32x32 PNG favicon
- `android-chrome-192x192.png` - Android icon (192x192)
- `android-chrome-512x512.png` - Android icon (512x512)

## App Icons (for PWA/Mobile)
- `manifest.json` - Web app manifest
- Various sized icons for different platforms

## Usage
These files are served from the root URL:
- `/favicon.ico`
- `/apple-touch-icon.png`
- etc.

Reference them in your HTML `<head>` section.
