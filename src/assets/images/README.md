# Images Directory

This directory contains image assets for the application.

## Logo Files
Place your logo files here:
- `logo.svg` - Primary logo (SVG format recommended)
- `logo.png` - Logo in PNG format (if needed)
- `logo-light.svg` - Light version of logo (for dark backgrounds)
- `logo-dark.svg` - Dark version of logo (for light backgrounds)

## Usage in Components
Import images in Svelte components like this:
```javascript
import logo from '../assets/images/logo.svg';
```

## Image Optimization
- Use SVG format when possible for scalability
- Optimize PNG/JPG files for web before adding
- Consider multiple resolutions for different use cases
